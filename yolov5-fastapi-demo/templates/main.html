이용 방법

1) 업로드할 때

1. Upload Images의 '파일 선택' 버튼을 누른 후
업로드할 사진을 개인 PC 환경에서 선택합니다.

2. YOLO Model의 최적 모형(best)이 선택된 것을 확인하고
파란색 결과 보기 버튼을 클릭합니다.

3. 결과를 확인합니다.

2) 직접 촬영할 때

1. 카메라가 켜지면 약품의 글씨가 쓰인 부분이
잘 보이도록 한 뒤 '사진 찍기' 버튼을 클릭합니다.

2. 우측 화면에 찍힌 사진을 확인하고 '사진 전송' 버튼을 클릭합니다.
다시 찍고 싶으면 '사진 찍기' 버튼을 다시 클릭합니다.

3. 결과를 확인합니다.
